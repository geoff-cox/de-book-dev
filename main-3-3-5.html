    Section 1.3 Variables

    Variables are an essential part of mathematics. They are used to represent unknown quantities in equations and functions. In differential equations, variables play a crucial role in understanding the relationship between the solution and the variable that the solution depends on. In this section, we will discuss the two types of variables that appear in differential equations: dependent and independent variables. We will also explore how to identify these variables in a differential equation and understand their roles in the context of solving differential equations.
    Recall from the previous sections, the solution to a differential equation such as
    \begin{equation} \frac{dy}{dx} + 2y = 3x^2,\tag{1} \end{equation}
    is some function, \(y(x)\text{.}\) This means that every differential equation inherently involves two variables: one representing the solution (i.e., \(y\)) and one representing the variable that the function depends on (i.e., \(x\)). Therefore, it is very important to understand the roles of these types of variables when working with differential equations.
    Dependent Variable
    The dependent variable is the function we are trying to find. It represents the solution to the differential equation. It is the \(y\) variable in equation (1).
    Independent Variable
    The independent variable is the variable on which the function depends. It’s the input of the function and the \(x\) variable in equation (1).

    Example 2.

    Identify the dependent and independent variables of the following DE:
    \begin{equation*} (1 - x)y'' - 4xy' + 5y = \cos x \end{equation*}
    Solution.
    Since \(y\) is the variable that has a derivative applied to it, it must be the dependent variable.
    We also know the following:
    • The other variable, \(x\text{,}\) must be the independent variable.
    • \(y\) represents the solution to this differential equation.
    • The solution, \(y\text{,}\) will be a function of \(x\text{.}\)

    Example 3.

    Identify the dependent and independent variables of the following DEs:
    \begin{equation*} y'' + xy' = \sin x, \quad \frac{dP}{dt} = P(P-10), \quad Q'' = 11Q \end{equation*}
    Solution.
    The first equation has dependent variable \(y\) and independent variable \(x\text{.}\)
    The second has dependent variable \(P\) and independent variable \(t\text{.}\)
    The dependent variable is \(Q\text{,}\) but we are not able to determine the independent variable without more information about what the DE represents.

    Example 4. Identify the Variables.

    Identify the dependent and independent variables in the differential equation
    \begin{equation*} \frac{dP}{ds} + \frac{P}{s^2} = 17s \text{,} \end{equation*}
    and state which variable represents the solution.
    Solution.
    Since \(\frac{dP}{ds}\) appears in the equation and is also the derivative of \(P\) with respect to \(s\text{,}\) the equation implies that \(P\) is a function of \(s\text{.}\) Therefore, the dependent variable is \(P\) and the independent variable is \(s\text{.}\) Moreover, the solution to this differential equation is the dependent variable, \(P\text{.}\)

    Example 5. Identify the Variables.

    Identify the dependent and independent variables in the differential equation
    \begin{equation*} u'' + t^2 \ u = 0 \text{,} \end{equation*}
    and state which variable represents the solution.
    Solution.
    Since \(\frac{dP}{ds}\) appears in the equation and is also the derivative of \(P\) with respect to \(s\text{,}\) the equation implies that \(P\) is a function of \(s\text{.}\) Therefore, the dependent variable is \(P\) and the independent variable is \(s\text{.}\) Moreover, the solution to this differential equation is the dependent variable, \(P\text{.}\)
    For the moment, consider the differential equation
    \begin{equation} x\frac{dy}{dx} - 2y = 4 - x.\tag{2} \end{equation}
    This differential equation has two variables, \(x\) and \(y\text{.}\) One of these variables represents the solution (a function) and the other represents the values that you plug into the function. In differential equations, it is important to know which one is which. For our example, \(y\text{,}\) is the solution and \(x\) represents the values we plug into \(y\text{.}\)
    It turns out that \(y(x) = x^2 + x - 2\) is a solution to (2) above.

    Definition 6. Dependent and Independent Variables.

    In a differential equation, the dependent variable is the variable that represents the unknown function (i.e., solution). The independent variable represents the variable that the unknown function depends on.

    Note 7. Useful Facts about Variables.

    • The independent variable may or not be present in a differential equation, but a dependent variable must be present.
    • If a variable has a derivative applied to it, it must be the dependent variable. The other variable present is the independent variable.
    • When we solve a differential equation, the dependent variable is the solution and it will be written as a function of the independent variable.
    • In general, if you were plotting a solution to a differential equation, the independent variable would correspond to the \(x\)-axis (horizontal) and the dependent variable would correspond to the \(y\)-axis (vertical).
