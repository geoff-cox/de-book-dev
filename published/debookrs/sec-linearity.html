    Section 2.2 Linearity

    Objectives

    After this section, you will be able to ...
    1. Identify if a differential equation is linear or nonlinear.
    2. Identify a specific nonlinear term in a differential equation.
    3. Anything else?
    WRITE SOME INTRODUCTORY TEXT...
    Every differential equation is also either linear or nonlinear.

    Definition 13.

    A differential equation is said to be linear if it can be written as
    \begin{equation} a_n(x)y^{(n)} + \cdots + a_2(x)y'' + a_1(x)y' + a_0(x)y = f(x).\tag{3} \end{equation}
    This definition is intended to represent the form of all possible linear differential equations, so it may seem a bit overwhelming. It may be easier to understand linearity based on what it is not, rather than what it is. So, let’s begin by describing characteristics of a DE that make it nonlinear.

    Identifying a nonlinear DE.

    A differential equation is nonlinear if any of the following are true about the dependent variable and/or its derivatives:
    \(\succ\)
    Has a power other than 1.
    Example:
    \(\DS y'' + y^2 = x^3\) is nonlinear because the dependent variable, \(y\text{,}\) is raised to the second power.
    \(\succ\)
    Is inside another function.
    Example:
    \(y'' + \ln(y') = e^x\) is nonlinear since the derivative, \(y'\text{,}\) is inside a logarithmic function.
    \(\succ\)
    Are multiplied together.
    Example:
    \(y'\cdot y'' + e^x y = y'\) is nonlinear since the first term is a product of \(y'\) and \(y''\text{.}\)

    Reading Questions Test your Understanding

    Let’s now look at an example of a linear differential equation.

    Example 14.

    Determine if the following differential equation is linear.
    \begin{equation*} y'' + \frac{y'}{x^2} + y = 17x \text{.} \end{equation*}
    Solution.
    This equation is almost in the form (with \(n=2\)) Definition 13. We just need to make a minor adjustment to see all the parts explicity,
    \begin{equation*} (1)y'' + \left(\frac{1}{x^2}\right)y' + (1)y = 17x \end{equation*}
    where (matching with Definition 13)
    \begin{equation*} \DS a_2 = 1, \,\, a_1 = \frac{1}{x^2}, \,\, a_0 = 1, \,\,\text{ and } \,\, f(x) = 17x \end{equation*}

    Exercises Linearity

    For each of the following differential equations, identify the independent and dependent variables, the order, and whether it is linear or nonlinear. If nonlinear, give one term in the expression that breaks the linearity.
    .

    1.

    \(\, \DS (1 - x)y'' - 4xy' + 5y = \cos x \)
    Answer.
    independent variable: \(x\)
    dependent variable: \(y\)
    2nd order
    linear

    2.

    \(\, \DS x \frac{d^3y}{dx^3} - \left( \frac{dy}{dx} \right)^4 + y = 0 \)
    Answer.
    independent variable: \(x\)
    dependent variable: \(y\)
    3rd order
    nonlinear, term: \(\DS \left( \frac{dy}{dx} \right)^4\)

    3.

    \(\, \DS t^5 y^{(4)} - t^3 y'' + 6y = 0 \)
    Answer.
    independent variable: \(t\)
    dependent variable: \(y\)
    4th order
    linear

    4.

    \(\, \DS \frac{d^2u}{dr^2} + \frac{du}{dr} + u = \cos(r+u) \)
    Answer.
    independent variable: \(r\)
    dependent variable: \(u\)
    2nd order
    nonlinear, term: \(\DS \cos(r+u)\)

    5.

    \(\, \DS \frac{d^2y}{dx^2} = \sqrt{1 + \left(\DS \frac{dy}{dx} \right)^2} \)
    Answer.
    independent variable: \(x\)
    dependent variable: \(y\)
    2nd order
    nonlinear, term: \(\DS \left(\DS \frac{dy}{dx} \right)^2\)

    6.

    \(\, \DS \frac{d^2R}{dt^2} = -\frac{k}{R} \)
    Answer.
    independent variable: \(t\)
    dependent variable: \(R\)
    2nd order
    nonlinear, term: \(\DS \frac{k}{R}\) or \(\DS \frac{1}{R}\)

    7.

    \(\, \DS (\sin \theta)y''' - (\cos \theta)y' = 2 \)
    Answer.
    independent variable: \(\theta\)
    dependent variable: \(y\)
    3rd order
    linear

    8.

    \(\, \DS \ddot{x} - \left( 1 - \frac{\dot{x}^2}{3} \right)\dot{x} + x = 0 \)
    Answer.
    independent variable: \(t\)
    dependent variable: \(x\)
    2nd order
    nonlinear, term: \(\DS \dot{x}^2\)

    9.

    \(\DS y\frac{dy}{dx} - 4y = x^6e^x\)
    Answer.
    independent variable: \(x\)
    dependent variable: \(y\)
    1st order
    nonlinear, term: \(\DS y\frac{dy}{dx}\)

    10.

    \(\DS \sin(x)\frac{dy}{dx} - 3y = 0\)
    Answer.
    independent variable: \(t\)
    dependent variable: \(x\)
    2nd order
    linear

    11.

    \(\DS \frac{dP}{dt}+2tP = P + 4t -2\)
    Answer.
    independent variable: \(t\)
    dependent variable: \(P\)
    1st order
    linear

    12.

    \(\DS \frac{dx}{dy} =x^2-3x \)
    Answer.
    independent variable: \(y\)
    dependent variable: \(x\)
    1st order
    nonlinear, term: \(\DS x^2\)
